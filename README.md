
# 🔋 Power Driver - Electronica de Potencia

**Una aplicación Android innovadora para control y gestión de sistemas de potencia**

---

## 📱 Descripción del Proyecto

**Power Driver** es una aplicación desarrollada en **Android Studio** que proporciona una interfaz intuitiva y potente para controlar, monitorear y gestionar dispositivos y sistemas de potencia. Diseñado para ingenieros, técnicos y entusiastas de la electrónica, esta app permite optimizar el rendimiento de tus sistemas.

### 🎯 Características Principales

- **Control en Tiempo Real**: Monitorea y controla sistemas de potencia desde tu dispositivo Android
- **Interfaz Intuitiva**: Diseño moderno y fácil de usar
- **Gestión de Dispositivos**: Conecta y administra múltiples dispositivos de potencia
- **Visualización de Datos**: Gráficos y estadísticas en tiempo real
- **Configuración Personalizable**: Ajusta parámetros según tus necesidades
- **Historial de Operaciones**: Registra todas las operaciones realizadas

---

## 🚀 Requisitos del Sistema

- **Android Mínimo**: Android 8.0 (API nivel 26)
- **Android Recomendado**: Android 11.0 o superior (API nivel 30+)
- **RAM**: Mínimo 2GB, recomendado 4GB
- **Almacenamiento**: 100MB disponibles

## 🛠️ Tecnologías Utilizadas

- **Gradle** - Sistema de construcción
- **Material Design** - Diseño de interfaz

## 📦 Instalación y Configuración

### Clonación del Repositorio

```bash
git clone https://github.com/Mate1592/ELECTRONICA-DE-POTENCIA-2026_1.git
cd ELECTRONICA-DE-POTENCIA-2026_1
```

### Construcción del Proyecto

```bash
./gradlew build
```

### Instalación en Dispositivo

```bash
./gradlew installDebug
```

## 📖 Guía de Uso

### Inicio Rápido

1. **Inicia la aplicación** en tu dispositivo Android
2. **Conecta tu dispositivo de potencia** vía Bluetooth o USB
3. **Accede al panel de control** principal
4. **Ajusta los parámetros** necesarios
5. **Monitorea las métricas** en tiempo real

### Pantallas Principales

#### 🏠 Dashboard
- Vista general del estado del sistema
- Indicadores de voltaje y corriente
- Botones de control rápido

#### ⚙️ Configuración
- Ajustes de parámetros de potencia
- Opciones de conexión
- Preferencias del usuario

#### 📊 Monitor
- Gráficos de voltaje y corriente
- Histórico de operaciones
- Exportación de datos

## 🐛 Troubleshooting

### Problema: La app no se conecta al dispositivo
**Solución**: Verifica que:
- El Bluetooth esté activado
- El dispositivo esté cerca
- La app tenga permisos de Bluetooth

### Problema: Datos incorrectos en lecturas
**Solución**: 
- Calibra el dispositivo desde Configuración
- Verifica las conexiones físicas
